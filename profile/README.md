## 愿景

> **构建一个开源、可演化、价值对齐的数字生命系统。**  
> 从单个 Agent 的工具使用，到大脑皮层间的信息关联，再到群体智能的涌现——我们探索 AI 系统从工具到生命的每一个阶段。
